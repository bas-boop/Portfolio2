Set Data to UI

This code snippet demonstrates how to update various UI elements with weather data. It takes in a WeatherResponse object and sets the text of different UI components based on the weather data received. The method updates temperature, cloud coverage, wind speed, degree, humidity, pressure, and visibility in the user interface.

private void UpdateUiWithData(WeatherResponse weatherData)
{
    if (weatherData.weather is { Length: > 0 })
    {
        main.text = weatherData.name + ": " + weatherData.weather[0].main;
        description.text = weatherData.weather[0].description;
    }

    temp.text = weatherData.main.temp + C;
    cloud.text = CC + weatherData.clouds.all + P;
    speed.text = WS + weatherData.wind.speed + METERS_PER_SECONDS;
    degree.text = D + weatherData.wind.deg + DEGREE_SYMBOL;
    humidity.text = HD + weatherData.main.humidity + P;
    pressure.text = PR + weatherData.main.pressure + H;
    visibility.text = V + weatherData.visibility + M;
}

API Key Loading Function

This code demonstrates how to load an API key from a configuration file. It reads a JSON file located in the streamingAssetsPath, extracts the openWeatherMapApiKey, and returns it. If the file is not found, an error is logged to the console.

public static string LoadApiKey()
{
    string path = Path.Combine(
        Application.streamingAssetsPath, 
        "Config.json"
    );
    
    if (File.Exists(path))
    {
        string json = File.ReadAllText(path);
        ApiConfig config = JsonUtility.FromJson(json);
        return config.openWeatherMapApiKey;
    }
    
    Debug.LogError("Configuration file not found.");
    return null;
}
