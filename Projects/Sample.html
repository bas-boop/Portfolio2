Getting Component

In this section, we delve into the process of retrieving components in Unity. The code snippet below illustrates a straightforward method to obtain a `MonoBehaviour` component attached to the GameObject. This approach is crucial for ensuring that the necessary components are correctly referenced and can be utilized effectively throughout the game. The method is simple yet effective, demonstrating how to handle component retrieval both during regular updates and late updates.

private MonoBehaviour something;
        
private void Update()
{
    something = GetComponent();
}

private void LateUpdate()
{
    if (something == null)
        something = GetComponent();
}
